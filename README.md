# metal-prices-inflation-tablue-python-Statistical-Analysis
Statistical analysis of Gold, Silver, Aluminium, Nickel, and Uranium prices using Python. The project explores price trends, inflation impact, correlation, linear regression, coefficient of variation (CV), and data visualization to identify market patterns and volatility.
Metal Prices & Inflation Analysis
Project Overview

This project analyzes the relationship between metal prices and inflation using statistical methods and data visualization. The study focuses on five major metals:

Gold
Silver
Aluminium
Nickel
Uranium

The objective is to identify price trends, measure volatility, evaluate inflation impact, and understand relationships between different metals through statistical analysis.

Objectives
Analyze historical metal price trends
Study the impact of inflation on metal prices
Compare price movements across different metals
Measure variability using the Coefficient of Variation (CV)
Examine relationships using Correlation Analysis
Predict relationships using Linear Regression
Present insights using visualizations

Key Findings
Gold and Silver prices show a strong positive relationship.
Nickel recorded the highest sales among all metals.
Uranium exhibited the highest volatility.
Aluminium showed the most stable price movement.
Metal prices are negatively correlated with general inflation but positively correlated with their own inflation trends.
